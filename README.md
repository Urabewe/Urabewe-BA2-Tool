# BA2 Manager

Small **Fallout 4 / Fallout 76 / Starfield** `.ba2` browser: open archives, extract files, pack textures (DDS), edit entries.

## Requires Python 3.8+ if you do not have python installed go here first:
https://www.python.org/downloads/

## Setup

1. Extract to a folder of your choosing.
2. Double‑click **`install.bat`** once (needs **Python 3.8+** on PATH). It creates a **`venv`** and installs **PyQt5** there.
3. Use **`launcher.bat`** to start the app.

You can move this whole folder anywhere; paths are relative to the batch files.

## Requirements

- Windows  
- Python 3.8 or newer (`python` or `py -3` on PATH)  
- Internet only for the first install (pip download)
